#PROJE AÇIKLAMASI
---------------------------------------------------------------------------------------
Axios ile JSON Server'a veri gönderme,düzenleme ve silme işlemlerinin yapıldığı bir CRUD uygulamasıdır.

#Kullanılan Teknolojiler
--------------------------------------------------------------------------------------
-Bootstrap
-React
-Axios
-React Router V6


#Uygulamayı Çalıştırmak için
------------------------------------------------------------------------------------
cd crud-app
npm run start:dev
